shortCuts

shortCuts shrinks your url to an unique five digit code. All links are stored in a relational database. Currently, we also store your shortened lijnks in local storage. You can delete all links by visiting dashboard.

Theoretically, there can exist only 52 C 5 [25,980,960] unique links. As the application is currently under development it is a good practice to delete all links at end of the month.

We also have RESTfull API that supports basic GET, POST requests. You can visit documentation for more information.
